# Guidelines:
1. Frok repo ini
2. Clone
3. Bikin folder nama kamu, bikin index.txt (boleh kosong atau di isi text terserah).
4. Commit
5. Push
6. Kirim PR ke repo ini

# Setelah PR di merge:
1. Update index.txt (atau bisa tambah file baru atau yang lainnya).
2. Commit
3. Push
4. Kirim PR ke repo ini
